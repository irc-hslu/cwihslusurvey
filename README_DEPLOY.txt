IEEEVR2026 real-study deploy package

Use stimuli.csv as the real experiment stimuli file. It has 35 rows/pairs.
Each pair uses two separate MP4s, so videos/ contains 70 real-study MP4 files.
Practice stimuli and videos are in practice_stimuli.csv and practice_videos/.
Metadata with source PLY paths, primitive counts, source PLY file sizes, category, and downsampling factor is in metadata/.
